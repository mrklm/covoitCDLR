Carte pour organiser des covoiturages pour l'équipe CDLR

Participants

Cliquez sur un nom pour renseigner les dates et les villes-étapes du trajet sélectionné.

    Aide et options

    Cette page permet de visualiser les trajets aller et retour de l'équipe vers le festival. Utilisez le sélecteur au-dessus de la carte pour basculer entre les allers et les retours.

    Dans la liste des participants, le bouton Renseigner ouvre le formulaire. Une personne peut proposer un covoit ou chercher une place. Seuls les trajets proposés sont dessinés sur la carte.

    Les données sont synchronisées avec Supabase lorsque la connexion est disponible. Une copie locale est conservée dans le navigateur pour garder l'application utilisable en cas de souci réseau.
