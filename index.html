    Aide et options

    Cette carte permet à l'équipe technique du festival Chalon dans la rue d'organiser les covoiturages.

    Cliquez sur une personne dans la liste, puis sur Renseigner, pour indiquer si elle propose un covoit ou cherche une place.

    Les trajets proposés sont affichés sur la carte avec une ligne colorée. Les personnes qui cherchent une place restent visibles dans la liste, mais aucun trajet n'est dessiné.

    Les trajets dont la date est passée ne sont plus affichés sur la carte. Les messages actifs défilent dans le bandeau sous l'en-tête.

    Le sélecteur au-dessus de la carte permet d'afficher les allers ou les retours.

    Pour l'aller, le départ est la ville de la personne par défaut et l'arrivée est Chalon-sur-Saône. Pour le retour, le départ est Chalon-sur-Saône et l'arrivée est la ville de la personne par défaut. Les villes modifiables peuvent être changées.

    Les données sont enregistrées automatiquement et partagées avec les autres utilisateurs.

    Le thème choisi est mémorisé dans ce navigateur et sera conservé à la prochaine ouverture de l'application.
